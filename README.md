## Project Overview
This project analyzes Swiggy food delivery data to understand delivery time,
ratings, and city-wise restaurant trends using Python and Power BI.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
- GitHub

## Dataset
- File: `swiggy.csv`

## Analysis Performed
- City-wise restaurant distribution
- Delivery time analysis
- Ratings vs delivery time
- Visual analysis using Python
- Interactive dashboard using Power BI

## Power BI Dashboard
The Power BI dashboard file can be downloaded and opened in Power BI Desktop.
